# Usage

Common keys to work with any project:

 Key                | Action            |
--------------------|----------------|
 <kbd>s-z</kbd> <kbd>s-/</kbd>  | Switch workgroups `wg-switch-to-workgroup`, `wg-switch-to-previous-workgroup`
 <kbd>s-]</kbd>     | Switch to a file in current repo (`helm-ls-git-ls`)
